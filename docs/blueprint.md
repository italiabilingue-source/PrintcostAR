# **App Name**: PrintCost AR

## Core Features:

- Parámetros de entrada: Captura todos los detalles relacionados con el cálculo de costos de la impresión 3D, incluyendo el costo del material, el tiempo de impresión, el consumo de electricidad, la mano de obra, la depreciación de la impresora, los costos de post-procesamiento y el margen de ganancia deseado.
- Desglose detallado: Proporciona cálculos paso a paso, lo que transparenta cómo se obtuvieron el costo final y el precio de venta.
- Cálculo final: Basado en las entradas del usuario, realiza cálculos y muestra el costo final y el precio de venta de la pieza impresa. Todos los campos de entrada deben ser editables y tomar valores dinámicos para permitir ajustes y recálculos fáciles.
- Moneda: Permite al usuario seleccionar la moneda en la que se mostrarán los costos.
- Estimaciones persistentes: Permite al usuario guardar diferentes estimaciones en una base de datos de Firestore para su posterior consulta y referencia.

## Style Guidelines:

- Color primario: Azul oscuro (#2C3E50) para una sensación profesional y confiable, inspirado en la ingeniería y la precisión.
- Color de fondo: Gris claro (#F4F4F8), proporcionando un telón de fondo limpio y neutro.
- Color de acento: Teal (#3498DB) para resaltar elementos importantes y llamadas a la acción.
- Fuente del cuerpo y del encabezado: 'Inter', una sans-serif de estilo grotesco para una apariencia moderna y objetiva; presentación limpia y fácil de leer, buena tanto para encabezados como para el cuerpo.
- Utiliza iconos de línea minimalistas para representar diferentes categorías de costos, proporcionando una experiencia de usuario limpia e intuitiva.
- Diseña un diseño limpio y estructurado, facilitando a los usuarios la introducción de datos y la comprensión del desglose de costos.
- Implementa animaciones y transiciones sutiles al actualizar y mostrar los resultados calculados, proporcionando una retroalimentación clara al usuario.
